## Intro to Python

Python uses: 
- quick and dirty scripting
- industrial scripts

Avoid Python for: writing efficient programs (since it's interpreted)
- note: Python modules can use code from other languages (e.g. C++) for optimization [e.g. PyTorch, TensorFlow, NumPy]
